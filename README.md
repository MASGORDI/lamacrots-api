# Lamacrots API

Servidor de datos macro para la web Lamacrots.

## Endpoints

- `GET /api/data` — Todos los datos (rates, VIX, bonds)
- `GET /api/rates` — Tipos de interés G10
- `GET /api/vix` — VIX en tiempo real
- `GET /health` — Estado del servidor

## Deploy en Railway

1. Sube estos 3 archivos a GitHub
2. Conecta el repo en Railway
3. Railway detecta automáticamente el proyecto Node.js
4. ¡Listo!
